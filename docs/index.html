Neohabitat

Neohabitat is a relaunch of the first massively multiplayer online roleplaying game, Lucasfilm's Habitat. Originally developed in 1985 by Lucasfilm Games (now LucasArts) for the Commodore 64, Habitat is considered the grandfather of modern MMORPGs. Neohabitat is an open-source project led by Randy Farmer, one of Habitat's creators, reviving the original Habitat world.
